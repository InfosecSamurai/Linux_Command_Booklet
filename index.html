File Operations

touch filename.txt - Create an empty file.

echo "Hello, World!" > filename.txt - Create a file with content.

nano filename.txt or vim filename.txt - Create and edit files in the terminal.

cat filename.txt - Display the content of a file.

less filename.txt - View content one page at a time.

more filename.txt - View content one screen at a time.

head filename.txt - Display the first ten lines of a file.

tail filename.txt - Display the last ten lines of a file.

diff file1.txt file2.txt - Compare two files line by line.

meld file1.txt file2.txt - A visual diff tool (requires installation).

ln -s target_file link_name - Create a symbolic link.

fdupes -r /path/to/directory - Find and delete duplicate files.

Directory Operations

mkdir new_directory - Create a new directory.

ls -l - List directory contents.

tree /path/to/directory - Display directory structure (requires installation).

cd /path/to/directory - Change directory.

rmdir directory_name - Remove an empty directory.

rm -r directory_name - Remove a directory and its contents.

chmod -R 755 directory_name - Change permissions recursively.

chown -R user:group directory_name - Change ownership recursively.

Viewing System Information

uname -a - Display system information.

sudo lshw - Display detailed hardware information.

lscpu - Display CPU architecture information.

hostname - Show or set the system's host name.

uname -r - Display the kernel version.

uptime - Show how long the system has been running.

dmesg - Print kernel ring buffer messages.

lsmod - Show the status of modules in the Linux kernel.

top - Display active processes.

htop - Interactive process viewer (requires installation).

vmstat 5 - Report virtual memory statistics.

free -h - Show memory usage.

df -h - Display disk space usage.

File Permissions

chmod 755 filename - Change file permissions.

chown user:group filename - Change file owner and group.

ls -l filename - View permissions of files.

umask 022 - Set default file permissions for new files.

Searching for Files and Text

find /path -name "filename" - Search for files in a directory hierarchy.

locate filename - Find files by name (requires updatedb).

grep "search_term" filename - Search for text within files.

grep -E "regex_pattern" filename - Use regex to search text.

Process Management

ps aux - Display running processes.

kill PID - Terminate a process.

killall process_name - Kill processes by name.

nice - Run a command with modified scheduling priority.

renice - Alter the priority of running processes.

pstree - Display a tree of processes.

jobs - List active jobs in the current shell.

fg %1 - Bring a job to the foreground.

bg %1 - Run a job in the background.

Networking Commands

ping example.com - Check connectivity to a host.

ifconfig - Display network configuration.

ip addr show - Show IP addresses.

netstat -tuln - Display network connections.

traceroute example.com - Trace route to a host.

route - Show/manipulate the IP routing table.

hostnamectl - Control the system hostname and related settings.

mtr - Combines ping and traceroute functionality.

iperf - Measure maximum TCP and UDP bandwidth performance.

wget --mirror - Download an entire website.

curl -O http://example.com/file.txt - Download files via curl.

wget http://example.com/file.txt - Download files via wget.

Package Management Commands

sudo apt-get install package_name - Install a package.

apt-cache search package_name - Search for packages.

snap - Package management system for installing and managing snaps.

sudo yum install package_name - Install a package.

sudo dnf install package_name - Next-gen package manager.

User Management

sudo adduser username - Add a new user.

sudo usermod -aG groupname username - Modify a user account.

sudo passwd username - Change a user's password.

whoami - Display the current user.

finger - Display information about users on the system.

chage - Change user password expiry information.

vipw - Safely edit the password file.

System Monitoring Tools

glances - Cross-platform system monitoring tool.

htop - Interactive process viewer.

iotop - Display I/O usage by processes.

pidstat - Report statistics by process.

tail -f /var/log/syslog - Monitor logs in real-time.

tcpdump - Dump traffic on a network.

nmap - Network exploration tool and security/port scanner.

socat - Multipurpose relay (socket cat).

Text Processing Commands

cut -d ':' -f 1 /etc/passwd - Remove sections from files.

sort file.txt - Sort lines of text files.

uniq -c sorted_file.txt - Report repeated lines.

awk '{print $1}' file.txt - Powerful pattern scanning language.

fmt - Simple text formatter.

paste - Merge lines of files.

pr - Convert text files for printing.

Archiving and Compression

tar -cvf archive.tar /path/to/directory/ - Archive files.

zip -r archive.zip directory_name - Zip files and directories.

tar -xvf archive.tar - Extract tar files.

unzip archive.zip - Extract zip files.

System Shutdown and Restart

sudo shutdown -h now - Shut down the system immediately.

sudo reboot - Restart the system.

sudo poweroff - Power off the system.

sudo shutdown -r +10 - Restart the system in 10 minutes.

Custom Commands

alias ll='ls -la' - Create an alias for a command.

unalias ll - Remove an alias.

nano myscript.sh - Create a new shell script.

Make it executable with chmod +x myscript.sh.

Run the script with ./myscript.sh.
